2.1 ASC文件前处理 — 计算表面高程

深度平均程序计算得到的h为滑体厚度或水深, 需要再此基础上加上基础表面的高程.

2.2 创建工程文件及模板图层

在进行处理之前, 需要手动创建一个空的工程项目以及一个套用的模板文件(lyrx图层文件)

  1. 创建任意一个项目
  2. 将任意一个asc文件导入工程中
  3. 转换为tiff栅格文件
  4. 创建山体阴影
  5. 修改符号系统、透明度、重采样类型等
  6. 右键 -> 共享 -> 保存为图层文件 -> 保存在模板文件夹内(根目录创建rft文件夹, 将文件保存在此文件夹内)

4. 深度图批量处理

与山体阴影的处理类似, 区别在于深度图不需要“创建山体阴影”步骤, 符号系统需要修改拉伸类型为“最大最小值”. 并且需要修改最大最小值至一个合适值(即总文件的最大最小值, 目的是统一颜色拉伸)

深度图只需要高度(深度, 厚度)即可, 因此不需要前处理. 只需要在处理山体阴影的程序基础上, 做以上修改即可.
