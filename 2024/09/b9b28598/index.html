ArcGis利用asc文件/栅格/点云作3D效果图教程

发表于2024-09-07|更新于2024-12-08|ArcGis
|字数总计:390|阅读时长:1分钟|阅读量:

1. 文件转换

ASC文件、栅格(tiff文件)和点云文件(ply文件)可以相互转换.

asc文件转tiff文件

将asc文件导入到ArcGis Pro或者ArcMap中, 右键文件选择导出栅格(数据). 格式为tiff.

ArcGis Pro:

若此处修改坐标系会出错, 就不定义坐标系. 在生成之后, 定义投影(教程在后面).

ArcMap:

tiff转ASC文件

ArcMap与之类似, 打开地理处理窗口 -> 搜索“栅格转ASCII” -> 输入相应的栅格与保存位置 -> 运行

与点云文件的互转可借助其它软件

我们最终得到具有坐标系定义的tiff文件, 如果原本就是tiff文件但没有坐标系定义. 需要进行定义投影(ArcGis Pro与ArcMap类似, 此后仅使用ArcGis Pro演示)

打开地理处理窗口 -> 搜索“定义投影” -> 选择栅格和坐标系 -> 运行

2. 使用tiff生成山体阴影

打开地理处理窗口 -> 搜索“山体阴影” -> 选择3D的山体阴影 -> 输入栅格和保存位置 -> 运行

3. 导入高程表面

右键高程表面 -> 选择从源创建表面图层 -> 选择之前的tiff文件(不是刚生成的山体阴影)

4. 设置浮动表面

右键山体阴影文件 -> 属性 -> 高程 -> 在自定义表面 -> 刚导入的表面图层(ts) -> 确认

5. 修改符号系统

修改符号系统至合适的颜色, 拉伸模式, 透明度等
